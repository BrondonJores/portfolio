# Presets Animation JSON

Ces fichiers sont compatibles avec l'import JSON de:
`Admin > Parametres > Animations > Presets Cinematiques > JSON de preset animation`.

## Comment importer

1. Ouvre un fichier `.json` de ce dossier.
2. Copie tout son contenu.
3. Colle dans le champ `JSON de preset animation`.
4. Clique `Importer JSON`.
5. Clique `Enregistrer`.

## Presets inclus

- `ultra-elementor-vibe.json`: style dynamique, hero fort, sprites humains tres presents.
- `cinematic-portfolio.json`: preset cinematic equilibre.
- `story-soft.json`: ambiance douce, transitions fluides et calmes.
- `arcade-energy.json`: style energique avec mouvements plus marques.
- `minimal-clean.json`: animation reduite, lisible et propre.
- `performance-focus.json`: compromis animation/charge pour site plus leger.
